# AI/ML Engineer Portfolio 🤖

A cutting-edge portfolio website with retro-futuristic terminal aesthetics, built for AI/ML engineers.

## 🎨 Features

- **Neural Network Theme**: Cyberpunk-inspired design with glitch effects
- **Animated Grid Background**: Dynamic matrix-style grid
- **CRT Screen Effects**: Authentic terminal feel with scanlines
- **Responsive Design**: Works perfectly on all devices
- **Smooth Animations**: Scroll-triggered reveals and hover effects
- **Custom Fonts**: IBM Plex Mono, Rajdhani, and Chakra Petch
- **Particle Effects**: Interactive mouse-tracking particles

## 🚀 Quick Deploy to Vercel

### Option 1: Using Vercel CLI

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to project directory:
```bash
cd path/to/portfolio
```

3. Deploy:
```bash
vercel
```

4. Follow the prompts:
   - Set up and deploy: Yes
   - Which scope: Your account
   - Link to existing project: No
   - Project name: ai-ml-portfolio (or your choice)
   - Directory: ./
   - Override settings: No

### Option 2: Using Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New" → "Project"
3. Import your Git repository (GitHub, GitLab, or Bitbucket)
4. Vercel will auto-detect the settings
5. Click "Deploy"

### Option 3: Drag and Drop

1. Go to [vercel.com](https://vercel.com)
2. Drag and drop the entire project folder
3. Vercel will deploy automatically

## 🎯 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-bg: #0a0e27;
    --accent-primary: #00f0ff;
    --accent-secondary: #ff00ff;
    --accent-tertiary: #00ff88;
}
```

### Fonts
Current fonts (AI/ML theme):
- Display: Chakra Petch
- Body: Rajdhani  
- Mono: IBM Plex Mono

To change, update the Google Fonts link in `index.html` and CSS variables.

### Content
Update your information in `index.html`:
- Personal details (lines 40-60)
- Education (lines 80-95)
- Skills (lines 100-200)
- Experience (lines 210-240)
- Projects (lines 250-350)
- Certifications (lines 360-400)
- Achievements (lines 410-440)
